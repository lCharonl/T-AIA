Stratégie d'optimisation

Façonner les récompenses pour accélérer l'apprentissage

On ajoute une récompense de potentiel (distance de Manhattan vers la cible) — une technique qui préserve la politique optimale (Ng et al., 1999) tout en guidant l'agent plus tôt.

« Convergence » = épisode où la moyenne glissante des pas passe durablement sous 20. Un coefficient trop fort peut distraire de l'objectif réel.

Synthèse pour le rapport

γ est décisif

Trop faible, le taxi ignore la prime finale (+20) et abandonne le passager. γ ≈ 0.97 fonctionne le mieux.

Le shaping accélère

Il fait converger ~40 % plus tôt sans changer la politique optimale — indispensable pour faire apprendre le DQN.

Le bon outil

Sur 500 états, la table tabulaire est exacte et efficace ; le DQN reste l'extension pour les grands espaces.
